# PROMPT: Mantenimiento de Práctica - Trapassato Prossimo

Usa este prompt para actualizar o expandir la pantalla de práctica del Pasado del Pasado.

**ARCHIVO OBJETIVO:** `app/pratica-trapassato-prossimo.tsx`

---

**PROMPT PARA LA IA:**

Actúa como un Desarrollador Senior de React Native. Tu tarea es mantener la pantalla de práctica del **Trapassato Prossimo**.

**REGLAS DE ACTUALIZACIÓN:**
1. **Ejercicios:** Situaciones donde una acción ocurrió antes que otra en el pasado.
2. **Estructura:** Imperfecto de Avere/Essere + Participio.
3. **Consistencia:** Mantén la reconstrucción de voz con `Speech`.

**EJEMPLO:**
```typescript
{
    id: 4,
    sentence: [{ text: 'Quando sono arrivato a casa, lei ' }, { hasInput: true, text: '', text_input: '' }, { text: ' (uscire).' }],
    answer: 'era uscita',
    translation: 'Cuando llegué a casa, ella ya había salido.',
    hint: 'Trapassato con movimento: era + uscita.'
}
```
