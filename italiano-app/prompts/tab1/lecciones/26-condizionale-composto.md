# PROMPT: Mantenimiento de Práctica - Condizionale Composto

Usa este prompt para actualizar o expandir la pantalla de práctica del Condicional Compuesto.

**ARCHIVO OBJETIVO:** `app/pratica-condizionale-composto.tsx`

---

**PROMPT PARA LA IA:**

Actúa como un Desarrollador Senior de React Native. Tu tarea es mantener la pantalla de práctica del **Condizionale Composto**.

**REGLAS DE ACTUALIZACIÓN:**
1. **Ejercicios:** Foco en acciones que "habrían pasado" pero no pasaron.
2. **Auxiliares:** Asegura el uso correcto de `Essere` o `Avere` en el condicional simple + participio.
3. **Tip:** Breve explicación de por qué se usa ese auxiliar.

**EJEMPLO:**
```typescript
{
    id: 5,
    sentence: [{ text: 'Noi ' }, { hasInput: true, text: '', text_input: '' }, { text: ' (mangiare) di più.' }],
    answer: 'avremmo mangiato',
    translation: 'Habríamos comido más.',
    hint: 'Avere -> avremmo + participio.'
}
```
