# PROMPT DE MANTENIMIENTO: Práctica 19 - Futuro Simple

Actúa como un Desarrollador Senior de React Native. Tu tarea es mantener o reconstruir la pantalla de práctica del Futuro Simple.

**ESTRUCTURA FILOSÓFICA:**
La pantalla `app/pratica-futuro.tsx` debe centrarse en la planificación y predicciones.

**REGLAS TÉCNICAS:**
1. **Lógica de Verificación:** Comparar el input del usuario con la respuesta conjugada correctamente (tener en cuenta los cambios de vocal temática -ARE -> -erò).
2. **Componentes:**
   - `ProgressBar` que use `Animated.timing`.
   - `TextInput` con `autoFocus` para fluidez.
   - `Speech` de Expo para pronunciación italiana.
3. **Estilo:** Icono `TrendingUp` para simbolizar el futuro. Colores coherentes con el tema del nivel B1.

**CONTENIDO SEMÁNTICO:**
Frases tipo: "Domani andrò...", "L'anno prossimo compreremo...", "Pioverà stasera...".
